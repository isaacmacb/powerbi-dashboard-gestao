# Dashboard Power BI 

Este projeto contém um dashboard desenvolvido no Power BI para análise de campanhas de marketing, número de consultas, e dados clínicos relevantes.

## 📊 Métricas disponíveis
- Impressões e cliques (Facebook, Google, Instagram)
- Conversões e custo por conversão
- Acompanhamento de pacientes (diabetes, glaucoma, etc.)

## 📁 Estrutura
- `Relatorio.pbix`: arquivo principal do Power BI
- `imagens/`: capturas de tela do relatório
- `dados-exemplo/`: exemplos de dados usados para testes

## 🔧 Ferramentas utilizadas
- Power BI
- PostgreSQL
- GitHub
